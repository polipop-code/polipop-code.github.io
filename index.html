Mis proyectos

Discord para la educación

HTML CSS Node.js Discord API Español

"Discord para la educación" Es un proyecto donde de la mano de la API de Discord en su versión 11.6.4 se desarrollo un BOT, el cual cuando se encuentra integrado a un servidor dentro de la aplicación Discord adquiere la funcionalidad de desplegar una plantilla de canales de texto y voz, esto por medio de una instrucción que se inserta dentro de un canal de texto.

Google clone

HTML CSS JavaScript English

"Google clone" Es un practico y funcional clon de la página de búsqueda principal de Google. Totalmente responsivo contiene un evento programado en JavaScript para ejecutar una busqueda en Google.com despues de ingresar algún texto en la barra de busqueda y preciónar la tecla "Enter".

MERN stack
